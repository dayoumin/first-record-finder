# KCI / RISS API 활용 신청 사유서 초안

API 키 발급 신청 시 활용목적 및 상세설명란에 사용할 수 있는 초안입니다. 공공성과 기관 내부 활용(기술동향 파악)을 강조하여 작성되었습니다.

---

## 📋 활용 목적 (공통)

**[활용분야]**
연구/학술 지원 및 공공 데이터 분석

**[서비스명 (가칭)]**
해양생물 기후변화 모니터링 및 학술동향 분석 시스템

## 📝 활용 내용 (상세 설명)

### 옵션 1: "공공성 + 내부 연구지원" 강조형 (추천)
> 기후변화로 인한 한반도 해역의 생물상 변화를 추적하기 위해 과거부터 현재까지의 문헌 기록을 분석하는 시스템을 개발 중입니다. KCI(또는 RISS)의 방대한 학술 데이터를 연동하여, 국내외 해양생물 출현 기록을 체계적으로 수집하고 분석하고자 합니다.
>
> 특히, 최신 연구 결과와 분류학적 변동 사항 등 기술 동향을 내부 연구진 및 직원들에게 신속하게 제공함으로써, 국가 생물다양성 관리 업무의 효율성을 높이고 과학적 데이터에 기반한 정책 수립을 지원하는 것이 주된 목적입니다. 수집된 데이터는 비영리 공익 목적의 연구 및 내부용 동향 파악 자료로만 활용됩니다.

### 옵션 2: "기술 동향 파악" 강조형
> 본 기관에서는 급변하는 해양 환경 및 생물 관련 기술 동향을 파악하기 위해 내부 직원용 정보 제공 시스템을 구축하고 있습니다.
>
> Open API를 통해 수집된 학술정보를 메타 분석하여, 최신 연구 트렌드와 주요 이슈를 시각화하여 제공함으로써 직원들의 직무 역량을 강화하고 연구 기획의 고도화를 도모하고자 합니다. 또한 국가적으로 시급한 기후변화 대응 연구에 필요한 기초 문헌 데이터베이스 구축에도 기여할 예정입니다.

---

## 💡 신청 팁
1. **활용 URL**: 개발 서버가 내부망(폐쇄망)인 경우, `http://localhost` 또는 `내부 IP`라고 적되, 비고란에 "기관 내부망 전용 시스템"이라고 명시하면 승인에 도움이 됩니다.
2. **트래픽**: 초기에는 '일 1,000~5,000건' 정도로 적정하게 신청하고, 사유에는 "특정 시점 배치를 통한 동향 분석"이라고 적으면 트래픽 급증 우려를 덜 수 있습니다.
